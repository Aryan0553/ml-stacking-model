# 🧠 Stacking Ensemble ML Classification
### Train vs Test Accuracy | Advanced Machine Learning

---

## 📌 Overview
This project implements an advanced **stacking ensemble machine learning model** to enhance classification performance and generalization. Multiple base learners are combined with a meta-model to reduce overfitting and achieve balanced train and test accuracy.

---

## 🎯 Objectives
- Build a non-traditional **ensemble ML model**
- Compare **training vs testing accuracy**
- Improve model robustness and reliability
- Visualize performance using industry-standard evaluation plots

---

## 📂 Dataset
- **Name**: Breast Cancer Wisconsin Dataset  
- **Source**: Scikit-learn built-in dataset  
- **Problem Type**: Binary Classification  

---
## ⚙️ Tech Stack
- **Programming Language**: Python  
- **Libraries Used**:
  - NumPy
  - Pandas
  - Scikit-learn
  - Matplotlib
  - Seaborn
- **Environment**: Jupyter Notebook  

---

## 🧠 Model Architecture

### 🔹 Base Models
- Logistic Regression  
- Decision Tree Classifier  
- K-Nearest Neighbors  

### 🔹 Meta Model
- Random Forest Classifier  

The predictions from base models are used as inputs to the meta-model to improve overall accuracy and generalization.

---

## 📊 Evaluation Metrics
- Train Accuracy vs Test Accuracy  
- Confusion Matrix

- Feature Importance  
- Prediction Probability Distribution
